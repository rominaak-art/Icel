# 👁️ Eyeless - Tarot Reading Blog

A dark-mode Persian tarot blog and request management system with The Fool card background.

## ✨ Features
- 🃏 **Tarot-themed design** with The Fool card background
- 🌙 **Dark mode** optimized for mystical reading experience
- 📝 **Request submission** - users send their questions
- 💬 **Reply system** - admin can respond to each request
- 📚 **Blog section** - share tarot articles and insights
- 💾 **LocalStorage** - all data saved in browser (no backend needed)
- 📱 **Fully responsive** - works on all devices

## 🚀 Live Demo
Open `index.html` in any browser to start.
